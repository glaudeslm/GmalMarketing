## 🚀 Como Publicar no GitHub

### Passo 1: Criar Repositório no GitHub

1. Acesse [GitHub.com](https://github.com)
2. Clique em "New Repository"
3. Preencha:
   - **Repository name**: `ai-email-marketing-assistant`
   - **Description**: `AI-powered email marketing tool with Groq API and CustomTkinter`
   - **Public** (importante!)
4. **NÃO** inicialize com README
5. Clique "Create repository"

### Passo 2: Fazer Upload dos Arquivos

**Opção A: Git (Recomendado)**

```bash
# Clone o repositório vazio
git clone https://github.com/SEU-USUARIO/ai-email-marketing-assistant.git
cd ai-email-marketing-assistant

# Copie todos os arquivos deste pacote para este diretório

# Configure git
git config user.name "Seu Nome"
git config user.email "seu-email@example.com"

# Faça commit
git add .
git commit -m "Initial commit: AI Email Marketing Assistant v1.0.0"

# Faça push
git branch -M main
git push -u origin main
```

**Opção B: GitHub Web Interface**

1. Vá até seu repositório
2. Click "Add file" → "Upload files"
3. Arraste e solte todos os arquivos
4. Clique "Commit changes"
